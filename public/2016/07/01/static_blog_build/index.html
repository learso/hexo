什么是静态博客

简单

无需数据库、评论功能，不需要不断的更新版本——只用关心你的博客内容。

静态

只用 Markdown (或 Textile)、Liquid、HTML & CSS 就可以构建可部署的静态网站。

博客形态

自定义地址、分类、页面、博客内容 以及 自定义的布局设计 都是系统中的一等公民。

常用的建静态博客的工具

  1. jekyll 最著名的静态博客引擎(ruby)
  2. octopress jekyll的一个框架，降低了使用难度
  3. hexo 使用node编写的静态博客生成器
  4. pelican Python编写的静态博客生成器(markdown, reST)
  5. liquidluck 国人编写, Python
  6. simple 可以在线编写静态博客

静态博客托管

  1. github pages GitHub是一个利用Git进行版本控制、专门用于存放软件代码与内容的共享虚拟主机服务（英语：Shared web hosting service）。
  2. coding.net pages Coding.net 是一个面向开发者的云端开发平台,提供git代码托管，免费的运行演示空间，代码质量分析，在线Web IDE。
  3. VPS 自己的vps当然可以想怎么弄就怎么弄，比如安装个nginx。
